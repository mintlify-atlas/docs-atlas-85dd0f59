Cruncher MCP is a zero-dependency scientific calculator server built on the Model Context Protocol (MCP). It gives any compatible AI assistant — Claude, Cursor, Goose, and more — access to 43 math tools with guaranteed accuracy, strict input validation, and infinite-loop protection via worker thread isolation.

## Why Cruncher?

AI language models routinely make arithmetic mistakes. Cruncher eliminates math hallucinations by routing calculations through a validated, sandboxed server that returns exact results.

<CardGroup cols={2}>
  <Card title="Zero dependencies" icon="leaf">
    Pure Node.js — no mathjs, no zod, no MCP SDK. Runs anywhere Node ≥ 18 is available.
  </Card>
  <Card title="Accurate decimals" icon="equals">
    Integer-scaling math means `0.1 + 0.2 === 0.3`. No IEEE 754 floating-point drift.
  </Card>
  <Card title="Safe expression eval" icon="shield-check">
    Strict regex whitelist — no `eval()`. Expressions are verified before execution.
  </Card>
  <Card title="Timeout protection" icon="clock">
    Worker threads with a 3-second kill switch prevent heavy calculations from freezing your assistant.
  </Card>
</CardGroup>

## Get started

<Steps>
  <Step title="Install via npm">
    Run Cruncher instantly with npx — no installation required:
    ```bash
    npx @slbdn/cruncher-mcp
    ```
  </Step>
  <Step title="Configure your AI client">
    Add Cruncher to your MCP client config. For Claude Desktop:
    ```json claude_desktop_config.json
    {
      "mcpServers": {
        "cruncher": {
          "command": "npx",
          "args": ["@slbdn/cruncher-mcp"]
        }
      }
    }
    ```
  </Step>
  <Step title="Start calculating">
    Ask your AI assistant any math question — Cruncher handles the computation:
    - "What is 15% of 240?"
    - "Calculate the standard deviation of [10, 12, 8, 14, 6]"
    - "Convert 50 miles to kilometers"
    - "Evaluate: sin(pi/4) + sqrt(16) + log10(100)"
  </Step>
</Steps>
