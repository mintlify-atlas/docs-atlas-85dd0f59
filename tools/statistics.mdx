## `median`

Calculates the median (middle value) of an array of numbers. For even-length arrays, returns the mean of the two middle values.

<ParamField body="numbers" type="number[]" required>
  Array of numbers. Does not need to be pre-sorted.
</ParamField>

```json
{ "numbers": [15, 22, 8, 41, 19, 30] }
// â†’ 20.5  (mean of 19 and 22)

{ "numbers": [3, 1, 4, 1, 5] }
// â†’ 3
```

## `variance`

Calculates the variance of an array of numbers. Supports both **sample variance** (n-1) and **population variance** (n).

<ParamField body="numbers" type="number[]" required>
  Array of numbers.
</ParamField>

<ParamField body="population" type="boolean" default="false">
  When `false` (default), calculates sample variance using (n-1) in the denominator â€” appropriate when the array is a sample from a larger population. When `true`, calculates population variance using (n) â€” appropriate when the array represents the entire population.
</ParamField>

```json
{ "numbers": [85, 92, 78, 90, 88] }
// â†’ 27.3  (sample variance, n-1)

{ "numbers": [85, 92, 78, 90, 88], "population": true }
// â†’ 21.84  (population variance, n)
```

## `std_dev`

Calculates the standard deviation of an array of numbers. Supports both **sample** (n-1) and **population** (n) mode.

<ParamField body="numbers" type="number[]" required>
  Array of numbers.
</ParamField>

<ParamField body="population" type="boolean" default="false">
  When `false` (default), calculates sample standard deviation (square root of sample variance). When `true`, calculates population standard deviation.
</ParamField>

```json
{ "numbers": [10, 12, 8, 14, 6] }
// â†’ 3.1623  (sample std dev)

{ "numbers": [10, 12, 8, 14, 6], "population": true }
// â†’ 2.8284  (population std dev)
```

## `percentile`

<Note>
  This tool requires `CRUNCHER_TOOL_SET=full`. It is not available in `minimal` or `standard` tiers.
</Note>

Returns the value at a given percentile in an array of numbers using linear interpolation.

<ParamField body="numbers" type="number[]" required>
  Array of numbers.
</ParamField>

<ParamField body="percentile" type="number" required>
  The percentile to calculate. Must be in the range `[0, 100]`.
</ParamField>

```json
{ "numbers": [10, 20, 30, 40, 50], "percentile": 75 }
// â†’ 40

{ "numbers": [23, 45, 12, 67, 89, 34, 56, 78, 90, 11], "percentile": 95 }
// â†’ 89
```

## Sample vs. population mode

| Mode | Formula | When to use |
| :--- | :--- | :--- |
| Sample (`population: false`) | Divide by nâˆ’1 | Array is a sample drawn from a larger dataset |
| Population (`population: true`) | Divide by n | Array represents the complete population |

When in doubt, use the default sample mode. It provides an unbiased estimate when working with a subset of data.
