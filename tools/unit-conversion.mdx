Cruncher MCP's `convert_unit` tool converts between 80+ units across 8 categories in a single call. Unit matching is case-insensitive, and temperature conversions use the correct non-linear formulas (C/F/K). This tool is available in the **standard** tier (the default).

---

## `convert_unit`

Converts a value from one unit to another within a supported category.

<ParamField body="value" type="number" required>
  The numeric value to convert.
</ParamField>

<ParamField body="category" type="string" required>
  The unit category. Must be one of: `length`, `weight`, `temperature`, `area`, `volume`, `time`, `speed`, `digital_storage`.
</ParamField>

<ParamField body="from" type="string" required>
  The unit to convert from. Case-insensitive (e.g., `"KM"` and `"km"` both work).
</ParamField>

<ParamField body="to" type="string" required>
  The unit to convert to. Case-insensitive.
</ParamField>

### Response format

```json
{
  "value": 5,
  "from": "mi",
  "to": "km",
  "result": 8.04672,
  "category": "length"
}
```

### Examples

```json
{ "value": 5, "category": "length", "from": "mi", "to": "km" }
// → { "result": 8.04672, ... }

{ "value": 100, "category": "temperature", "from": "C", "to": "F" }
// → { "result": 212, ... }

{ "value": 1, "category": "digital_storage", "from": "GB", "to": "MB" }
// → { "result": 1024, ... }

{ "value": 60, "category": "speed", "from": "mph", "to": "km/h" }
// → { "result": 96.56064, ... }
```

## Notes on usage

- **Case-insensitive matching**: `"KM"`, `"Km"`, and `"km"` are all equivalent.
- **Temperature is non-linear**: Celsius, Fahrenheit, and Kelvin conversions use the correct offset formulas, not scaling factors.
- **Cross-category conversion is not supported**: `from` and `to` units must belong to the same `category`.
